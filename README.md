# AWS Services Hands-on Projects

## About This Repository
This repository contains hands-on AWS service implementations
based on real DevOps learning and practical scenarios.

## Services Covered
- EC2
- S3
- IAM
- VPC
- CloudWatch (if applicable)

## Purpose
To help beginners understand AWS services through
simple, real-world use cases instead of theory.

## Community Focus
This repository is maintained to support learners
who are starting their AWS journey.
